## とりあえず動かしてみる
- - - 
インタープリターの起動とHello,World
```
$ python
>>> hello = "Hello, World"
>>> hello
'Hello, World'
```

インタープリターの終了
```
>>> exit()
$
```

## 電卓がわりに使う
- - -
四足演算
```
>>> a = 6
>>> b = 2
>>> a + b
8
>>> a - b
4
>>> a * b
12
>>> a / b
3.0
```
ビットシフト
```
>>> 1 << 3
8
```

## 10進数
10進数の数字を扱う(数字⇄文字)
```
>>> 123
123
>>> str(123)
'123'
>>> int('123')
123
>>> '%08d' % 123
'00000123'

```

---

## 16進数
16進数の数字を扱う  
基本は0xXXの形で表現できる
* 10 ⇄ 16

```
>>> hex(123)
'0x7b'
>>> int('0x7b', 16)
123
>>> int('7b', 16)
123
```
